netupvimで使用しているHTTPダウンロードモジュールを取り出し、コマンドライン ツールとしたものです。PROXY部分の妥当性評価に使用します。ソースコードをsrc に収録しています。コマンドラインより次のコマンドを実行してください。
> dotest.bat PROXY_URL_OR_IP:PORT
これでexecute.logと、ダウンロードに成功すればdownload.logという2つのファ イルが作成されます。download.logが作成された場合、同梱のsample.logと中身 を比べてください。次のような場合execute.logとdownload.logを添付して村岡
